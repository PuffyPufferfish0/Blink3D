# Blink3D
A 3D modeling software designed for triangulation and textures. 

make command:

```
g++ main.cpp glad.c Mesh.cpp Camera.cpp imgui/*.cpp -I./include -I./imgui -I/usr/include/SDL2 -o main -lSDL2 -ldl
```

run main after make

```
./main
```
